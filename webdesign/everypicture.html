Every Picture Project

in this project i was tasked to

create a webpage that documents your personal archive

using visual prompts for exploration, narrative, and storytelling to create effective and engaging interactions.

Program

Figma, Photoshop, Visual Studio Code

Team

Solo Project

phase one

every picture project

In the first part of my project, I first decided what I wanted to create my webpage archive about. Growing up, my family often visited national parks and I began collecting various stamps to commemorate each visit. I designed a mockup of what I envisioned my webpage to look like: a stampbook, just like the one I use to collect my stamps. I then proceeded to code my project in Visual Studio Code.

The final version of my project, also found HERE, included a booklet with flippable pages that featured some of my favorite stamps and a short description of the national park it came from when clicked upon.

phase two

final project revisions

For our final project assignment, we were asked to pick one of our projects and heavily revise it. I chose this project to work on, as it was a project I felt that could use a lot of improvement and further test my coding abilities.

The two main things I wanted to work on in my revision was image inclusions and overlays. I wanted to improve the visual design of my project by adding a background image behind the notebook to help tell the story. Additionally, I planned to design the internal notebook pages more so that they were less plain and more integrated with the notebook theme. I used Photoshop to create a mockup of what I visualized these changes to look like in the final version.

Secondly, I wanted to make better use of my space by using overlays. To do this, I planned to expand the overlay to incorporate more images and a longer description of my experiences at each place. I also wanted to make the images on these overlays more interactive to increase user interactions.

Next, I asked two people to critique my project and identify what could be improved for the final version of my project. My key findings from these critiques aligned with my plans, including: improving visual design of notebook pages and imagery, changing my cursor type to indicate something clickable, and removing unnecessary buttons.

I took my previous project and implemented my proposed changes as best I could. The result, which can be found HERE, was highly successful. I was able to incorporate many of the changes I wanted to, and as a result my final project became my best and favorite work from this class.
